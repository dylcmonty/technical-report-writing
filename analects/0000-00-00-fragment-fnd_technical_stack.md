# Technical stack

FND’s current stack includes Python, Flask, REST APIs, PostgreSQL and SQL, pandas for ETL work, Bash, Docker and Docker Compose, NGINX, AWS EC2 and Route 53, Linux, Git, HTML, CSS, and JavaScript.

I also use C++ where lower-level system work is appropriate. The product architecture is moving toward a SQL-backed core with portable web and desktop deployment, structured data schemas, and modular service adapters.
