# Fruitful Network Development

I operate Fruitful Network Development as a cloud-hosted software effort focused on tools for local agriculture and small-business systems. I have run deployments on AWS with NGINX reverse proxy and Docker Compose, built Flask APIs and admin tooling, and developed Python pipelines for reporting and communications workflows. The larger goal behind that work is to make digital infrastructure more transparent, locally controlled, and useful to the people who depend on it.
