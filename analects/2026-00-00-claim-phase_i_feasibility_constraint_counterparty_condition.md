# Phase I Feasibility Constraint Counterparty Condition

The counterparty constraint is grocery-side non-worsening
The model cannot only improve the farm side in isolation. It must also test whether the grocery or hub side remains economically viable. A coordination scenario is only meaningful if local sourcing improves the farm position while keeping delivered cost, service level, or margin from becoming materially worse for the buyer side.
## Boundaries
- This does not require immediate superiority on every metric.
- It does require an explicit buyer-side constraint.
