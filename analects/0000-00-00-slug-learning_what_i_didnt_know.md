# Learning What I Didnt Know

One of the things I am most proud of is the way my software project’s data schema forced me into multidisciplinary learning. I was trying to learn what I did not know I did not know while building the code and schema, which led me into ideas like NP-complete problems, ontology, and self-organizing systems. I value that kind of research because it changes how I understand the structure of a problem, not just how I code around it.
