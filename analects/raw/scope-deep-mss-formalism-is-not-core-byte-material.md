# Deep MSS formalism is not core-byte material
Slug: scope-deep-mss-formalism-is-not-core-byte-material
Kind: scope
Question: What material should stay out of the main byte set?
Answer:
The deep formalism about infinite recombination, delimiter logic, and a universal language of information should not sit in the core byte set for the proposal. That material belongs in a separate whitepaper or appendix unless it is translated into a bounded technical artifact that Phase I can actually test.
Boundaries:
- This is not a rejection of the theory.
- It is a scope decision about what belongs in the main working set.
