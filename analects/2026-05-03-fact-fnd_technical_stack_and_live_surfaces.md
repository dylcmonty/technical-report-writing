# FND technical stack and live surfaces

## Fact

I currently build FND with Python, Flask, REST APIs, PostgreSQL, SQLite where appropriate, pandas, BeautifulSoup, Bash, Docker, NGINX, Gunicorn, systemd, AWS EC2, Route 53, Linux, Git, HTML, CSS, JavaScript, and Rust tooling for local agent workflows.

The live MyCite/FND system includes a portal shell, `/portal` routing, `/portal/api/v2/...` admin and API surfaces, SQL-backed authority views, NGINX ingress, localhost-bound internal services, oauth2-proxy, Keycloak, and service deployment tooling.
