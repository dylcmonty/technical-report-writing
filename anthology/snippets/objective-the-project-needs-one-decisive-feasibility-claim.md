# The project needs one decisive feasibility claim
Slug: objective-the-project-needs-one-decisive-feasibility-claim
Kind: objective
Question: What question cracks the seal on the rest of the proposal?
Answer:
The project needs one Phase I feasibility claim that, if shown to be true, makes the rest of the effort rational to fund. Everything else should be downstream of that claim. The proposal becomes coherent only when there is a defined decision, a minimum demonstration, and a clear failure condition.
Boundaries:
- This is a discipline rule for the proposal, not a market claim.
- This forces scope instead of expansion.
