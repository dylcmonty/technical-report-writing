# Semantic diff is the most defensible primitive
Slug: architecture-semantic-diff-is-the-most-defensible-primitive
Kind: architecture
Question: Which technical primitive is the strongest candidate for a Phase I proof point?
Answer:
Semantic diff is a defensible primitive because it focuses on whether differing farm records, plans, or inventory descriptions can be mapped into a shared structure without requiring uniform software. That is more testable than broad claims about a universal language of information and closer to the coordination problem the proposal actually needs to solve.
Boundaries:
- This does not discard the broader architecture.
- It narrows the Phase I proof point to something testable.
